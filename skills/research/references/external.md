# External Evidence

1. Define freshness boundary + authoritative source class.
2. Search primary sources: official docs/specs/standards/changelogs/source repos/papers/vendor pages.
3. Use secondary sources only for discovery or unresolved context; label them.
4. Verify claim against source content + publication/update date + applicable version.
5. Cite direct URL beside every material claim; preserve conflicting sources.
6. Finish when the decision is answered or the missing primary proof is explicit.

- Current/high-stakes fact → browse; cached model memory alone = insufficient.
- Quote minimally; summarize within source limits.
