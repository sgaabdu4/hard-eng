# Web quality

Read [web-quality.md](references/web-quality.md) for matching React/Next/TS
quality work. Existing project scripts, design tokens, components, and CI own
the implementation contract.

This optional skill consolidates static analysis, React diagnostics, and
performance guidance behind one front door. It does not auto-install tools,
auto-fix semantic findings, add another review stage, or launch a model.
